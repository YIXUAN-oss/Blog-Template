# 📖 日常随笔使用指南

> 本指南将帮助你快速使用和自定义日常随笔页面

## 📝 快速开始

编辑 `docs/diary/README.md` 文件，添加你的随笔内容。

## ✍️ 添加新随笔

### 方式 1：在同一文件中添加

在"最近更新"部分的 `diary-list` 中添加新的随笔：

```markdown
<div class="diary-item">
  <div class="diary-date">2025-01-15</div>
  <h3>你的随笔标题</h3>
  <div class="diary-content">
    <p>
      你的随笔内容...
    </p>
    <p>
      可以写多段内容。
    </p>
    <p><em>—— 你的名字</em></p>
  </div>
</div>
```

### 方式 2：创建独立文件（推荐）

在 `docs/diary/` 目录下创建独立的 Markdown 文件：

```markdown
---
title: 随笔标题
date: 2025-01-15
categories:
  - 日常随笔
tags:
  - 生活感悟
  - 思考
---

# 随笔标题

你的随笔内容...

## 段落 1

内容...

## 段落 2

内容...
```

然后在 `README.md` 中链接到这些文件，或者让它们自动显示在博客列表中。

## 📅 随笔格式

### 基本结构

```markdown
<div class="diary-item">
  <div class="diary-date">日期</div>
  <h3>标题</h3>
  <div class="diary-content">
    <p>内容段落 1</p>
    <p>内容段落 2</p>
    <p><em>—— 署名（可选）</em></p>
  </div>
</div>
```

### 日期格式

建议使用 `YYYY-MM-DD` 格式：

```markdown
<div class="diary-date">2025-01-15</div>
```

### 标题

使用简洁有吸引力的标题：

```markdown
<h3>🌟 开启博客之旅</h3>
<h3>💭 关于学习的思考</h3>
<h3>🚀 一个小目标</h3>
```

## 🏷️ 分类标签

在"分类标签"部分，可以添加或修改标签：

```html
<span class="tag">💻 编程思考</span>
<span class="tag">📚 读书笔记</span>
<span class="tag">🎯 目标计划</span>
<span class="tag">✨ 生活感悟</span>
<span class="tag">🌈 随想杂谈</span>
```

## 💭 生活感悟

在"生活感悟"部分，可以添加你喜欢的名言或感悟：

```markdown
<div class="quote-box">
  <blockquote>
    "你的感悟或名言"
  </blockquote>
  <p class="quote-author">—— 作者</p>
</div>
```

## 🎨 自定义样式

### 修改主题色

在文件末尾的 `<style scoped>` 标签中修改：

```css
.diary-item {
  border-left: 4px solid #你的颜色; /* 修改左边框颜色 */
}

.tag {
  background: #你的颜色; /* 修改标签背景色 */
}
```

### 修改布局

可以调整随笔卡片的样式：

```css
.diary-item {
  padding: 30px; /* 调整内边距 */
  margin-bottom: 40px; /* 调整间距 */
}
```

## 📝 内容建议

### 可以写什么？

- **技术思考**：学习新技术的心得体会
- **生活感悟**：日常生活中的思考和感悟
- **读书笔记**：阅读书籍后的感想
- **目标计划**：个人目标和计划
- **随想杂谈**：各种随想和杂谈

### 写作建议

1. **真实自然**：记录真实的想法和感受
2. **定期更新**：保持一定的更新频率
3. **分类清晰**：使用标签进行分类
4. **简洁明了**：内容简洁，重点突出

## 🔧 常见问题

### Q: 如何添加图片？

在随笔内容中使用图片：

```markdown
<img src="/images/diary/your-image.jpg" alt="描述" style="border-radius: 8px;" />
```

### Q: 如何添加链接？

使用 Markdown 链接语法：

```markdown
[链接文本](https://example.com)
```

### Q: 如何修改日期格式？

修改 `diary-date` 的显示格式，或在 CSS 中调整：

```css
.diary-date {
  font-size: 0.9em; /* 调整字体大小 */
}
```

### Q: 随笔太多怎么办？

可以考虑：
1. 分页显示
2. 按年份分类
3. 使用标签筛选
4. 创建独立的随笔文件

## 💡 最佳实践

### 内容组织

1. **按时间排序**：最新的随笔放在最前面
2. **使用标签**：方便分类和查找
3. **定期更新**：保持内容的新鲜度

### 写作技巧

1. **真实记录**：记录真实的想法和感受
2. **简洁明了**：内容简洁，重点突出
3. **有始有终**：每篇随笔有明确的主题
