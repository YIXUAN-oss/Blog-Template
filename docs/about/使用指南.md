# 📖 关于页面使用指南

> 本指南将帮助你快速自定义关于页面

## 📝 快速开始

编辑 `docs/about/README.md` 文件，修改以下内容：

### 1. 修改个人信息

#### 头部信息

```markdown
<h2>你的名字</h2>
<p class="tagline">你的标签 | 你的身份 | 你的特点</p>
```

#### 个人简介

在"个人简介"部分，替换为你的介绍：

```markdown
## 💫 个人简介

你好！我是[你的名字]，[你的职业/身份描述]。

[在这里写一段关于你自己的介绍]
```

### 2. 更新技术栈

在"技术栈"部分，修改为你熟悉的技术：

```markdown
### 后端开发
<div class="skill-tags">
  <span class="skill-tag">☕ Java</span>
  <span class="skill-tag">🍃 Spring Boot</span>
  <!-- 添加或删除技术标签 -->
</div>
```

**添加技术标签：**
```html
<span class="skill-tag">🔧 技术名称</span>
```

### 3. 修改学习理念

在"学习理念"部分，可以修改为你的学习理念：

```markdown
<div class="philosophy-item">
  <h3>🎯 你的理念标题</h3>
  <p>你的理念描述</p>
</div>
```

### 4. 更新年度目标

在"2025 年目标"部分，修改为你的目标：

```markdown
<div class="goal-item">
  <input type="checkbox" disabled> 你的目标内容
</div>
```

### 5. 添加项目经历

在"项目经历"部分，添加你的项目：

```markdown
### 🌐 项目名称
项目描述。

**技术栈**: 使用的技术栈

**链接**: [项目地址](https://github.com/your-username/project)
```

### 6. 更新联系方式

在"联系方式"部分，修改为你的联系方式：

```markdown
<div class="contact-card">
  <div class="contact-icon">📧</div>
  <h3>Email</h3>
  <a href="mailto:your-email@example.com">your-email@example.com</a>
</div>
```

**添加更多联系方式：**
```html
<div class="contact-card">
  <div class="contact-icon">🐦</div>
  <h3>Twitter</h3>
  <a href="https://twitter.com/your-username" target="_blank">@your-username</a>
</div>
```

### 7. 修改兴趣爱好

在"兴趣爱好"部分，修改为你的兴趣爱好：

```markdown
- 📚 **阅读** - 你的阅读偏好
- ✍️ **写作** - 你的写作内容
```

### 8. 更新座右铭

在"座右铭"部分，修改为你喜欢的名言：

```markdown
<div class="motto-box">
  <blockquote>
    "你的座右铭"
  </blockquote>
  <p class="motto-author">—— 作者</p>
</div>
```

## 🎨 自定义样式

### 修改主题色

在文件末尾的 `<style>` 标签中，可以修改颜色：

```css
.profile-header {
  background: linear-gradient(135deg, #你的颜色1 0%, #你的颜色2 100%);
}

.contact-card a {
  color: #你的主题色;
}
```

### 修改布局

可以调整各个部分的布局，例如：

```css
.philosophy-box {
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); /* 调整列宽 */
}
```

## 📸 图片配置

### 头像图片

确保 `docs/.vuepress/public/avatar.png` 文件存在，或者修改路径：

```markdown
<img src="/avatar.png" class="avatar" alt="你的名字" />
```

## 💡 最佳实践

### 内容建议

1. **个人简介**：简洁明了，突出特点
2. **技术栈**：列出你熟悉和正在学习的技术
3. **项目经历**：选择有代表性的项目展示
4. **联系方式**：提供常用的联系方式

### 更新频率

- **技术栈**：定期更新，反映当前技能
- **年度目标**：每年年初更新
- **项目经历**：有新项目时及时添加

## 🔧 常见问题

### Q: 如何添加新的联系方式？

在"联系方式"部分添加新的 `contact-card`：

```html
<div class="contact-card">
  <div class="contact-icon">📱</div>
  <h3>微信</h3>
  <span>your-wechat-id</span>
</div>
```

### Q: 如何修改头像？

1. 将头像图片放到 `docs/.vuepress/public/` 目录
2. 确保文件名为 `avatar.png`（或修改代码中的路径）

### Q: 如何添加更多技术标签？

在对应的技术栈部分添加：

```html
<span class="skill-tag">🔧 技术名称</span>
```

### Q: 如何修改页面样式？

在文件末尾的 `<style scoped>` 标签中修改 CSS 样式。
