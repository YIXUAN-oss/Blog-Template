# 📖 博客文章使用指南

> 本指南将帮助你快速掌握如何编写和管理博客文章

## 🚀 快速开始

### 创建第一篇博客文章

1. **在 `docs/blogs/` 目录下创建 Markdown 文件**

```bash
docs/blogs/我的第一篇文章.md
```

2. **添加 Front Matter 配置**

在文件开头添加：

```markdown
---
title: 我的第一篇文章
date: 2025-01-01
categories:
  - 技术
  - 前端
tags:
  - JavaScript
  - Vue
author: 你的名字
---
```

3. **编写文章内容**

```markdown
# 我的第一篇文章

文章内容...
```

## 📝 Front Matter 配置详解

### 必需字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `title` | 文章标题 | `title: 我的第一篇文章` |
| `date` | 发布日期 | `date: 2025-01-01` |
| `categories` | 分类（数组） | `categories: ['技术', '前端']` |
| `tags` | 标签（数组） | `tags: ['JavaScript', 'Vue']` |

### 可选字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `author` | 作者 | `author: 你的名字` |
| `description` | 文章描述 | `description: 这是一篇关于...` |
| `sticky` | 置顶（数字，越大越靠前） | `sticky: 1` |
| `sidebar` | 是否显示侧边栏 | `sidebar: false` |
| `article` | 是否作为文章 | `article: true` |

### 完整示例

```markdown
---
title: 深入理解 Vue 3 响应式原理
date: 2025-01-15
categories:
  - 技术
  - 前端
  - Vue
tags:
  - Vue
  - JavaScript
  - 响应式
author: 你的名字
description: 本文深入探讨 Vue 3 的响应式系统实现原理
sticky: 1
---
```

## ✍️ 文章编写

### 标题层级

使用清晰的标题层级组织内容：

```markdown
# 一级标题（文章标题）

## 二级标题（章节）

### 三级标题（小节）

#### 四级标题（子小节）
```

### 代码块

使用代码块展示代码，并指定语言：

````markdown
```javascript
function hello() {
    console.log('Hello, World!');
}
```
````

**支持的语言：**
- `javascript` / `js`
- `typescript` / `ts`
- `python` / `py`
- `java`
- `bash` / `shell`
- `html` / `css`
- 等等...

### 图片使用

#### 方式 1：本地图片

将图片放在 `docs/blogs/attachments/` 目录：

```markdown
<img src="/blogs/attachments/image.jpg" alt="图片描述" style="border-radius: 8px;" />
```

#### 方式 2：图床图片（推荐）

使用图床服务，推荐 GitHub + jsDelivr：

```markdown
<img src="https://cdn.jsdelivr.net/gh/username/repo@master/path/to/image.jpg" alt="图片描述" style="border-radius: 8px;" />
```

**图床服务推荐：**
- **GitHub** + jsDelivr（免费、稳定）
- **GitCode**（国内访问快）
- **Gitee**（国内访问快）
- **其他图床服务**

#### 图片样式建议

```markdown
<!-- 基础样式 -->
<img src="..." alt="描述" style="border-radius: 8px;" />

<!-- 带缩放 -->
<img src="..." alt="描述" style="zoom: 50%; border-radius: 8px;" />

<!-- 居中显示 -->
<div style="text-align: center;">
  <img src="..." alt="描述" style="border-radius: 8px;" />
</div>
```

### 引用

使用引用突出重要内容：

```markdown
> 这是一段引用文字
> 可以多行
```

#### 有序列表


### 链接

### 强调


## 🖼️ 图片管理

### 目录结构

```
docs/blogs/
├── attachments/          # 博客图片目录
│   ├── image1.jpg
│   ├── image2.png
│   └── ...
├── 文章1.md
├── 文章2.md
└── ...
```

### 图片处理工具

项目提供了图片处理脚本 `process_blog_images.py`：

```bash
# 处理所有博客文章
python process_blog_images.py

# 处理指定目录
python process_blog_images.py docs/blogs/

# 处理单个文件
python process_blog_images.py docs/blogs/文章名.md
```

**功能：**
- ✅ 自动为图片添加圆角样式
- ✅ 在相邻图片之间添加分割线
- ✅ 将 Markdown 格式图片转换为 HTML 格式

### 图片优化建议

1. **压缩图片**：使用工具压缩图片大小
2. **使用 WebP**：现代浏览器支持，体积更小
3. **使用图床**：减少仓库体积，提升加载速度
4. **添加 alt 文本**：提升 SEO 和可访问性

## 🏷️ 分类和标签

### 分类（Categories）

分类用于组织文章，建议使用：

- **技术分类**：前端、后端、移动端、DevOps 等
- **内容分类**：教程、实战、总结、思考 等
- **技术栈分类**：Vue、React、Java、Python 等

**示例：**
```yaml
categories:
  - 技术
  - 前端
  - Vue
```

### 标签（Tags）

标签用于标记文章关键词，建议：

- 使用具体的技术名称
- 使用相关的概念
- 3-5 个标签为宜

**示例：**
```yaml
tags:
  - Vue
  - JavaScript
  - 响应式
  - 原理
```

## 🔧 常见问题

### Q: 文章不显示在博客列表？

**解决方案：**
1. 检查文件是否在 `docs/blogs/` 目录
2. 检查 Front Matter 配置是否正确
3. 确保 `date` 字段格式正确（YYYY-MM-DD）
4. 重启开发服务器

### Q: 图片不显示？

**解决方案：**
1. 检查图片路径是否正确
2. 使用绝对路径：`/blogs/attachments/image.jpg`
3. 确保图片文件存在
4. 检查文件名大小写

### Q: 如何添加代码高亮？

使用代码块并指定语言：

````markdown
```javascript
// 代码内容
```
````

### Q: 如何添加数学公式？

VuePress 支持 LaTeX 数学公式：

```markdown
$$
E = mc^2
$$
```

### Q: 如何添加目录？

文章会自动生成目录，也可以手动添加：

```markdown
[[toc]]
```

### Q: 如何添加警告/提示框？

```markdown
::: tip 提示
这是一个提示
:::

::: warning 警告
这是一个警告
:::

::: danger 危险
这是一个危险提示
:::
```
