# 1.3 第一个示例

## 创建项目

让我们创建一个简单的示例项目：

```bash
# 创建项目目录
mkdir my-first-project
cd my-first-project

# 初始化项目
tool init
```

## 编写代码

创建一个简单的程序：

```javascript
// main.js
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet('World'));
```

## 运行示例

运行程序：

```bash
node main.js
```

**输出结果：**
```
Hello, World!
```

## 代码说明

### 函数定义

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}
```

- `function`：定义函数的关键字
- `greet`：函数名称
- `name`：函数参数
- `return`：返回结果

### 函数调用

```javascript
console.log(greet('World'));
```

- `greet('World')`：调用函数并传入参数
- `console.log()`：输出结果到控制台
