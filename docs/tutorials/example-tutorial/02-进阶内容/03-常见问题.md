# 2.3 常见问题

## 问题 1：安装失败

### 问题描述

安装工具时出现错误。

### 解决方案

1. **检查系统要求**
   ```bash
   # 检查系统版本
   uname -a  # Linux/macOS
   systeminfo  # Windows
   ```

2. **检查权限**
   ```bash
   # 确保有管理员权限
   sudo command  # Linux/macOS
   # 以管理员身份运行  # Windows
   ```

3. **清理缓存**
   ```bash
   # 清理安装缓存
   tool clean-cache
   ```

## 问题 2：配置不生效

### 问题描述

修改配置后，更改没有生效。

### 解决方案

1. **检查配置文件路径**
   ```bash
   # 确认配置文件位置
   tool config --path
   ```

2. **验证配置格式**
   ```bash
   # 验证配置文件
   tool config --validate
   ```

3. **重启服务**
   ```bash
   # 重启服务使配置生效
   tool restart
   ```

## 问题 3：性能问题

### 问题描述

程序运行缓慢。

### 解决方案

1. **检查资源使用**
   ```bash
   # 查看资源使用情况
   top  # Linux/macOS
   tasklist  # Windows
   ```

2. **优化代码**
   ```javascript
   // 使用更高效的算法
   // 减少不必要的计算
   // 使用缓存
   ```

3. **调整配置**
   ```yaml
   # 增加资源限制
   memory: 2GB
   cpu: 4
   ```

## 问题 4：依赖冲突

### 问题描述

不同依赖包版本冲突。

### 解决方案

1. **检查依赖版本**
   ```bash
   # 查看依赖树
   tool deps --tree
   ```

2. **更新依赖**
   ```bash
   # 更新到兼容版本
   tool update --resolve-conflicts
   ```

3. **使用虚拟环境**
   ```bash
   # 隔离依赖环境
   tool env create
   tool env activate
   ```

## 调试技巧

### 启用调试模式

```bash
# 启用详细日志
tool --debug

# 设置日志级别
tool --log-level=debug
```

### 查看日志

```bash
# 查看日志文件
tail -f /var/log/tool.log  # Linux/macOS
type tool.log  # Windows
```
