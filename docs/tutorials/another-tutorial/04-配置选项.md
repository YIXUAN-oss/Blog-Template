# 配置选项

## 配置文件

创建 `config.json` 文件：

```json
{
  "apiKey": "your-api-key",
  "timeout": 5000,
  "retries": 3,
  "debug": false
}
```

## 配置项说明

### apiKey

- **类型**：`string`
- **必需**：是
- **说明**：API 密钥，用于身份验证

```javascript
{
  apiKey: 'your-api-key'
}
```

### timeout

- **类型**：`number`
- **默认值**：`5000`
- **说明**：请求超时时间（毫秒）

```javascript
{
  timeout: 10000  // 10秒
}
```

### retries

- **类型**：`number`
- **默认值**：`3`
- **说明**：失败重试次数

```javascript
{
  retries: 5
}
```

### debug

- **类型**：`boolean`
- **默认值**：`false`
- **说明**：是否启用调试模式

```javascript
{
  debug: true
}
```

## 环境变量

也可以通过环境变量配置：

```bash
export EXAMPLE_API_KEY=your-api-key
export EXAMPLE_TIMEOUT=5000
export EXAMPLE_DEBUG=true
```

## 配置优先级

1. 环境变量
2. 配置文件
3. 默认值
