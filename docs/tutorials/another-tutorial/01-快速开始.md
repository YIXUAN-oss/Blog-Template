# 快速开始

## 5 分钟快速上手

本指南将帮助你在 5 分钟内快速上手。

## 安装

```bash
npm install example-package
```

## 基本使用

### 第一步：导入

```javascript
import { Example } from 'example-package';
```

### 第二步：创建实例

```javascript
const example = new Example({
    option1: 'value1',
    option2: 'value2'
});
```

### 第三步：使用

```javascript
example.doSomething();
```

## 完整示例

```javascript
import { Example } from 'example-package';

// 创建实例
const example = new Example({
    apiKey: 'your-api-key',
    environment: 'production'
});

// 执行操作
const result = await example.process({
    input: 'Hello, World!'
});

console.log(result);
```
